#  Door Hinge

## About

Door Hinge is my toy dating app project. It is a native iOS app using SwiftUI
with a Go backend. This is my first attempt at a full-stack mobile application

My friends said this app was a terrible idea to make, but I'm making it
anyways to act all cool and defiant.
